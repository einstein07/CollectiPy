randn_like (line 382) — the function that uses prange:
```python
@staticmethod
def randn_like(y, sigma, inv_sqrt_n):
    out = np.empty_like(y)
    for i in prange(y.size):   # <-- prange loop present here
        out[i] = np.random.normal(0.0, sigma) * inv_sqrt_n
    return out
```
NOTE: randn_like is a plain @staticmethod (NOT decorated with @njit). The prange call
in randn_like only runs in parallel if randn_like itself is compiled with @njit(parallel=True).
Currently there is NO @njit decorator on randn_like at all — the prange loop runs
sequentially. The fix is to add @njit(parallel=True) to randn_like.

Existing @njit usage in the file:
```python

```
No other @njit functions exist in the file. The fix is scoped to randn_like only.
