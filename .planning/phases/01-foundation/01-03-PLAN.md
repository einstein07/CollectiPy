Key methods:
- mf.step() — advances by one call to compute_dynamics(integration_time, dt)
- mf.run(steps) — calls step() N times, returns trajectory array shape (N, num_neurons)
- mf.neural_ring — current state vector, shape (num_neurons,)
- mf.theta — neuron angles array, shape (num_neurons,), linspace(-pi, pi, 50, endpoint=False)
- compute_center_of_mass(z, theta) — returns float angle in [-pi, pi]
- mf.reset() — clears state back to zeros

To test bump formation: provide num_targets=1, set sensory input via step() call with
target_angles=[angle] and target_qualities=[quality]. Do NOT call compute_sensory_map
separately — step() calls it internally.

To trigger divergence for TEST-02 regression: create MeanFieldSystem with dt=10.0,
sigma=0.0, beta=0.0, u=6.0 — the Euler step is unstable at this dt. Call step()
in a loop and expect RuntimeError within a few steps.

    Test 2 (test_bump_forms_from_uniform_ic):
    - Create MeanFieldSystem(num_neurons=50, sigma=0.0, dt=0.1, integration_time=50.0,
        u=6.0, beta=1.0, num_targets=1, kappa=20.0)
    - Set initial state to uniform: mf.neural_ring = np.ones(50) * 0.01
    - Target angle: 0.0 radians. Call step with target_angles=[0.0], target_qualities=[1.0]
    - Call mf.run(steps=30, target_angles=[0.0], target_qualities=[1.0])
    - Assert np.max(mf.neural_ring) - np.min(mf.neural_ring) > 0.1 (bump is non-uniform)

    Test 3 (test_single_target_response):
    - Create MeanFieldSystem(num_neurons=100, sigma=0.0, dt=0.1, integration_time=50.0,
        u=6.0, beta=1.0, num_targets=1, kappa=20.0)
    - Reset to zero IC
    - Run 50 steps with target at pi/4 radians, quality=1.0:
        mf.run(steps=50, target_angles=[np.pi/4], target_qualities=[1.0])
    - Compute bump angle: angle = compute_center_of_mass(mf.neural_ring, mf.theta)
    - Assert abs(angle - np.pi/4) < np.pi/8 (bump is within 22.5 degrees of target)
