```python
existing = [d for d in os.listdir(abs_base_path) if d.startswith("config_folder_")]

```

Replace it with this ATOMIC version:
```python
abs_base_path = os.path.join(os.path.abspath(""), base_path)
os.makedirs(abs_base_path, exist_ok=True)
folder_id = 0
while True:
    candidate = os.path.join(abs_base_path, f"config_folder_{folder_id}")
    try:
        os.makedirs(candidate, exist_ok=False)
        self.config_folder = candidate
        break
    except FileExistsError:
        folder_id += 1
```

Key points:
- `os.makedirs(candidate, exist_ok=False)` is atomic on POSIX: it raises FileExistsError if
  the directory already exists, with no race window between check and creation.
- The retry loop increments `folder_id` until an unclaimed name is found.
- The old `raise Exception(f"Error config folder ... already present")` is removed — the
  loop handles that case transparently.
- The subsequent `with open(...config.json...)` line and all code below the replaced block
  must remain unchanged.
- Do NOT change the SpaceDataHandling class or any other method.
- Do NOT add any import — `os` is already imported on line 12.
